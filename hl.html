Building the Higher Lower Game

The Game - How It Works

The higher lower game is a knowledge-based strategy game about guessing whether one thing has a higher or lower monthly search volume than another. You start with two items, and if you guess correctly, you gain a point and move to the next round. If you guess incorrectly, the round ends and your score resets. The item with the higher search volume stays for the next round while the lower item is replaced.

The Game - How I Coded It

The logic I used to make the game was an if-else statement. I stored the search volumes for several items in a list, then randomly selected the first two. If the user guessed correctly, for example when A has a higher search volume than B, then B would be swapped with a new random item, like C. If the guess was wrong, the game ended and the score was displayed.
