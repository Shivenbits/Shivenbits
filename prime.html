Why prime numbers become rarer — A short proof using the Prime Number Theorem

Equations are typeset with MathJax.

Statement (Prime Number Theorem)

Let \(\pi(x)\) denote the number of primes \(\le x\). The Prime Number Theorem (PNT) says

\[\pi(x)\sim\frac{x}{\log x}\qquad(x\to\infty).\]

From PNT to "primes get rarer"

Divide both sides of the asymptotic relation by \(x\) to obtain the proportion of integers up to \(x\) that are prime:

\[\frac{\pi(x)}{x}\sim\frac{1}{\log x}.\]

Since \(\log x\to\infty\) as \(x\to\infty\), the right-hand side tends to \(0\). Therefore

\[\lim_{x\to\infty}\frac{\pi(x)}{x}=0.\]

Interpreting this probabilistically: the chance that a randomly chosen large integer near \(x\) is prime is about \(1/\log x\), which goes to zero as \(x\) grows. Hence primes become rarer.

Key probability approximation

Using PNT we get

\[\mathbb{P}({n\text{ is prime}})\approx\frac{\pi(n)}{n}\approx\frac{1}{\ln n}.\]

Example: \(n=10^6\)

\[\frac{1}{\ln(10^6)}\approx\frac{1}{13.8155}\approx 0.0723\quad(\approx7.23\%).\]

Example: \(n=10^9\)

\[\frac{1}{\ln(10^9)}\approx\frac{1}{20.7233}\approx 0.0483\quad(\approx4.83\%).\]

Intuition (divisibility perspective)

A number is prime if it avoids divisibility by any smaller prime. As \(n\) grows, there are more small primes that could divide it, so the constraint set grows. The PNT encapsulates this cumulative effect by showing the density of primes drops like \(1/\log n\).

Remark (elementary bounds)

If you prefer an elementary (weaker) route, Chebyshev proved that for large \(x\) there are positive constants \(c_1,c_2\) with

\[c_1\frac{x}{\log x}\le\pi(x)\le c_2\frac{x}{\log x},\]

which also implies \(\pi(x)/x\to0\) as \(x\to\infty\).
